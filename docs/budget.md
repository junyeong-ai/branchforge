# Budget

Budget management controls spending behavior for agent execution.

## Main Components

- `BudgetTracker`
- model pricing support
- exceed policies
- tenant-aware budget management

## Supported Policies

- stop before the next request
- warn and continue
- switch to a fallback model

## Scope

Budget management is about cost control. It complements token and context tracking but does not replace it.
