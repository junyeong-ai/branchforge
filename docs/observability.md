# Observability

The runtime includes local metrics and tracing primitives, with optional OpenTelemetry support.

## Built-in Areas

- request and token metrics
- tool execution metrics
- latency histograms
- span-based tracing helpers

## Scope

Observability is designed to support agent execution analysis and integration with larger monitoring systems.

## Optional OpenTelemetry

Use the `otel` feature when you want to integrate with OpenTelemetry exporters.
