# Output Styles

Output styles customize how the runtime assembles assistant-facing response guidance.

## What Output Styles Control

- response tone
- explanation depth
- coding-instruction retention
- custom style-specific guidance

## Built-in Styles

- `default`
- `explanatory`
- `learning`

## Sources

Output styles can come from:

- built-in definitions
- file-based definitions

## Design Rule

Output styles shape assistant behavior, but they are not part of authentication or provider transport logic.
