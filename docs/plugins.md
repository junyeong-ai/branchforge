# Plugins

Plugins package reusable runtime resources behind a namespace.

## Plugin Scope

Plugins can contribute:

- skills
- subagents
- hooks
- MCP server definitions

## Namespace Rule

Plugin resources are namespaced to avoid collisions with project or built-in resources.

## Practical Use

Plugins are useful when the same workflows or integrations must be reused across multiple projects.
